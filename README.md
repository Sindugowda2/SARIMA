This project is a Crop Forecasting Dashboard built using Python and Streamlit.
It helps users analyze agricultural data, make crop predictions, and view insights through an interactive web dashboard.

The application includes:
Dashboard view
Crop Prediction page
About page
Contact page

Features
1. Interactive Streamlit web interface
2. Crop prediction using machine learning
3. Dashboard with visual insights
4. Simple navigation using sidebar
5. Beginner-friendly and easy to run

Technologies Used
Python 3
Streamlit
Pandas
NumPy
Scikit-learn
Matplotlib
